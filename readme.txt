插件更多详细介绍和安装：[https://github.com/sy-records/random-look](https://github.com/sy-records/random-look)

== Installation ==

1. Upload the folder `random-look` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's all
